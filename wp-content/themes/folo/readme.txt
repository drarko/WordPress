== Description ==

Folo is a business + portfolio theme where you can showcase work samples, products and services. Homepage features a circular slider to highlight products and services. The theme comes with various grid and list layout options (21 choices in total). It allows you to build custom pages with specific categories. On top of that, there is an optional sorting/filtering navigation where your visitors can filter the post types.

== Installation ==

1. Upload the `folo` folder to the `/wp-content/themes/` directory in your server
2. Activate the theme in the `Appearance\Themes` screen in the WordPress Admin
